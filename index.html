Backbone Map Widget

Plots pins on google map with a list of names displayed along side.

Download this project as a .zip file Download this project as a tar.gz file

A simple backbone map widget. It reads data in JSON format and plots the places as pins on the map. Its easy to customize and can be of great help for people wanting to learn backbonejs.
